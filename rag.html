RAG Stages

1. Retrieve

Documents are cleaned, chunked, converted into embeddings, stored in a vector database, and searched semantically.

2. Augment

Retrieved chunks are injected into the prompt to enrich the LLM context.

3. Generate

The LLM uses query + retrieved context + instructions to generate a grounded answer.
